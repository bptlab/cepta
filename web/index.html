About

Realtime monitoring for everyone

CEPTA allows you to start monitoring any transports that can be tracked. Supply chains are critical and CEPTA can warn you about upcoming problems so you can reschedule.

  • Free and Open Source
  • Event based
  • Early warning of potential problems
  • Highly scalable
  • Prototype UI and User Management included
